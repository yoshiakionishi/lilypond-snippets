# Old Versions

In this folder you will find the versions of the Cookbook as compiled with a stable version for the last time before migrating to a newer stable version. 

For example, the `Version January 15, 2026` is the last version of the Cookbook to be compiled with LilyPond `2.24.4`, that version is found here. The subsequent version of the Cookbook will be made with LilyPond `2.26.0`.
